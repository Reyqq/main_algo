1.1. Модуль Утилит (utils)#

Модуль вспомогательных утилит.

Содержит функции для сериализации, преобразования типов данных, масштабирования, вычисления вероятностей, работы с файлами и данными.

utils.save_pickle(obj: Any, path: str) → None[исходный код]#

Сохраняет объект Python в файл с использованием pickle.

Параметры:

  • obj (Any) – Объект Python для сохранения.

  • path (str) – Путь к файлу, в который будет сохранен объект. Расширение файла обычно .pkl или .pickle.

utils.load_pickle(path: str) → Any[исходный код]#

Загружает объект Python из файла pickle.

Параметры:

path (str) – Путь к файлу pickle для загрузки.

Результат:

Загруженный объект Python.

Тип результата:

Any

utils.standard_scaler(array: ndarray) → ndarray[исходный код]#

Стандартизирует массив (вычитает среднее и делит на стандартное отклонение).

Применяет стандартизацию (Z-score normalization).

Параметры:

array (np.ndarray) – Входной NumPy массив для стандартизации. Декоратор @lst_to_array преобразует список в массив, если нужно.

Результат:

Стандартизированный массив. Имеет среднее ~0 и стд. отклонение ~1.

Тип результата:

np.ndarray

utils.get_probabilities(array: ndarray, method: str = 'default', beta: float = 1.0) → ndarray[исходный код]#

Преобразует массив значений в массив вероятностей различными методами.

Параметры:

  • array (np.ndarray) – Входной массив значений (например, оценки или fitness). Декоратор @lst_to_array преобразует список в массив.

  • method (str, optional) – Метод преобразования. Доступные методы: „default“: Простое нормирование делением на сумму. „rank“: Ранговое нормирование (чем выше ранг, тем выше вероятность). „softmax“: Softmax преобразование с параметром beta. „uniform“: Равномерное распределение (эквивалентно softmax с beta=0). По умолчанию «default».

  • beta (float, optional) – Параметр для метода „softmax“, контролирующий «резкость» распределения. По умолчанию 1.0.

Результат:

Массив вероятностей, сумма элементов которого равна 1.

Тип результата:

np.ndarray

utils.select_method(config_proba: Dict[str, float]) → str[исходный код]#

Выбирает метод случайным образом на основе заданных вероятностей.

Параметры:

config_proba (Dict[str, float]) – Словарь, где ключи - названия методов, а значения - их вероятности выбора. Сумма вероятностей должна быть равна 1.

Результат:

Случайно выбранное название метода.

Тип результата:

str

utils.get_action_idxes(n_obs: int, proba: float) → ndarray[исходный код]#

Генерирует индексы для выполнения действия на основе вероятности.

Для каждого из n_obs наблюдений генерируется случайное число. Если оно меньше proba, индекс этого наблюдения добавляется в результат.

Параметры:

  • n_obs (int) – Общее количество наблюдений (или потенциальных действий).

  • proba (float) – Вероятность выполнения действия для каждого наблюдения (от 0 до 1).

Результат:

Массив целых чисел - индексы наблюдений, для которых

должно быть выполнено действие.

Тип результата:

np.ndarray

utils.get_file_rowcols(path: str) → Tuple[int, int][исходный код]#

Получает количество строк и столбцов для файлов Parquet или HDF5.

Примечание: Работает только для стандартных табличных структур. Для HDF5 предполагается структура „table/columns“ и одинаковое количество строк во всех столбцах (проверяется по первому столбцу).

Параметры:

path (str) – Путь к файлу (.parquet или .hdf5).

Результат:

Кортеж (количество строк, количество столбцов).

Тип результата:

Tuple[int, int]

utils.get_chunks(data: ndarray | DataFrame, n_jobs: int, axis: int = 0) → List[ndarray | DataFrame][исходный код]#

Разбивает данные (NumPy массив или Pandas DataFrame) на части (chunks).

Параметры:

  • data (Union[np.ndarray, pd.DataFrame]) – Входные данные для разбиения.

  • n_jobs (int) – Количество частей (chunks), на которые нужно разбить данные. Должно быть положительным целым числом.

  • axis (int, optional) – Ось, по которой происходит разбиение. 0 - разбиение по строкам (по умолчанию). 1 - разбиение по столбцам (доступно только для DataFrame). По умолчанию 0.

Результат:

Список частей данных. Тип элементов

списка соответствует типу входных данных.

Тип результата:

List[Union[np.ndarray, pd.DataFrame]]
